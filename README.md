-Segitiga<br>
-Segitiga Siku-Siku<br>
-Persegi<br>
-Persegi Panjang<br>
-Layang-Layang<br>
-Jajargenjang<br>


<b><i>Bangun Ruang:</i></b><br>
-Balok<br>
-Kubus<br>
-Limas Segitiga<br>
-Prisma<br>
